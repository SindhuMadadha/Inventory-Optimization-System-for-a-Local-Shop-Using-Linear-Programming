Inventory Optimization System for a Local Shop Using Linear Programming
Project Overview
This project is an Inventory Optimization System developed using Python and the PuLP optimization library in Google Colab. The primary objective of the project is to determine the optimal quantity of products that a local shop should stock in order to maximize profit while operating under resource constraints.

Inventory management is an important aspect of retail businesses. Shop owners often face limitations such as storage capacity, budget restrictions, and product availability. Making inventory decisions without proper analysis can lead to overstocking, understocking, and reduced profitability. This project demonstrates how linear programming can be used to solve such business problems efficiently.

The system models a profit maximization problem involving two products: Milk and Bread. Decision variables are defined to represent the number of packets of each product to stock. An objective function is then created to maximize total profit based on the profit contribution of each product.

Business constraints are incorporated into the model to represent real-world limitations. These constraints restrict the total quantity of products that can be stocked and ensure that available resources are not exceeded. The optimization problem is solved using the PuLP library, which automatically finds the best possible solution that satisfies all constraints while maximizing profit.

After solving the model, the system provides the optimal number of Milk and Bread packets to stock. The results indicate that stocking 50 Milk packets and 50 Bread packets yields the highest achievable profit of 1750 units while satisfying all defined constraints. This demonstrates how mathematical optimization techniques can support business decision-making and improve operational efficiency.

The project introduces the fundamentals of operations research, linear programming, decision variables, objective functions, and constraint-based optimization. It serves as a practical example of applying optimization techniques to inventory management and business planning problems.
